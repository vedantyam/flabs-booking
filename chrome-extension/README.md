# FLABS Demo Booking Chrome Extension

## How to install (one time per SP):
1. Open Chrome → go to chrome://extensions
2. Turn ON "Developer mode" (top right toggle)
3. Click "Load unpacked"
4. Select this `chrome-extension` folder
5. Extension is installed! Pin it to toolbar.

## How to use:
1. Make sure you are logged into TeleCRM in Chrome
2. Open the FLABS booking page
3. Book a demo — TeleCRM will update automatically
4. Click the extension icon to see status

## Note:
- Must be logged into TeleCRM in Chrome for automation to work
- Uses YOUR TeleCRM login — updates show YOUR name in activity
